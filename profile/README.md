## Pckk-iRIC

Pckk-iRIC は、**iRIC のソルバー**と、iRIC の **入出力ファイル／データ処理などの周辺ツール**を管理する Organization です。


### この Organization にあるもの
- **iRIC ソルバー**（計算実行ファイル・SolverDefinition など）
- **周辺ツール**（iRIC の入出力データ変換、前処理／後処理、検証、可視化補助 など）


### リポジトリ命名ルール（重要）
iRIC ソルバーは、次の命名規約で管理しています：

- **ソルバー**：`iric-solver-<solvername>`

> 例：`iric-solver-rriresultviewer` / `iric-solver-cat2d`

- Releases（ある場合）：配布物や変更点はこちら
